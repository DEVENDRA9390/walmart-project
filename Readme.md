# Walmart Sales Data Analysis

## Project Overview
This project analyzes Walmart sales data using Python to uncover sales trends, customer behavior, product performance, and business insights. The analysis includes data cleaning, exploratory data analysis (EDA), and visualizations to support data-driven decision-making.

## Objectives
- Analyze overall sales performance.
- Identify top-performing product categories.
- Understand customer purchasing behavior.
- Discover sales trends across branches and cities.
- Generate actionable business insights.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Information
The dataset contains Walmart sales transaction records, including:
- Invoice ID
- Branch
- City
- Customer Type
- Gender
- Product Line
- Unit Price
- Quantity
- Total Sales
- Payment Method
- Rating

## Project Workflow

### 1. Data Cleaning
- Checked for missing values.
- Removed duplicates.
- Corrected data types.
- Prepared data for analysis.

### 2. Exploratory Data Analysis (EDA)
- Sales by Product Line
- Revenue by Branch
- Customer Type Analysis
- Payment Method Distribution
- Monthly Sales Trends
- Gender-wise Purchasing Patterns

### 3. Data Visualization
- Bar Charts
- Line Charts
- Histograms
- Pie Charts
- Heatmaps

## Key Insights
- Identified the highest revenue-generating product categories.
- Analyzed branch-wise sales performance.
- Discovered customer preferences and buying patterns.
- Evaluated payment method usage.
- Highlighted factors contributing to higher sales.
