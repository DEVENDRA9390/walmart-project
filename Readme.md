**"Walmart Sales Analysis Dashboard - SQL | Python | Power BI"**

**Project Overview**



This end-to-end Data Analytics project analyzes Walmart sales data using SQL, Python, and Power BI. The objective is to uncover sales trends, identify top-performing stores, evaluate holiday impacts, and understand how economic factors influence business performance.



The project follows the complete Data Analytics lifecycle, including data cleaning, exploratory data analysis (EDA), SQL business analysis, dashboard development, and business recommendations.
